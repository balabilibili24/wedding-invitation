# 分享兼容性与页面加载设计

## 目标

在不改动背景音乐行为和页面视觉内容的前提下，提高微信分享抓取兼容性，并移除地图脚本对首屏的竞争请求。

## 范围

- `og:image` 使用带版本参数的自定义域名直连 URL，并声明 `secure_url`、MIME 类型和替代文本。
- 每张内容图片声明其真实宽高；首张合影优先加载，其余继续惰性加载。
- 高德地图 API 在地图容器进入视口前才注入；脚本失败时保留地址文字和空白地图容器。
- 不改变 `audio` 标签、背景音乐预加载或自动播放逻辑。

## 验收

- 静态检查确认 OG 标签完整、版本 URL 使用 HTTPS，且不存在同步高德脚本标签。
- 39 张本地内容图片都有与文件一致的 `width` 与 `height`。
- 嵌入 JavaScript 可以被 Node 编译，所有本地图片引用存在。
